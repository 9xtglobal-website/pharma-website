## File format

- `.jpg` is preferred (smaller, faster to load)
- `.png` and `.webp` also work — just rename your files to match the names above
- Square aspect ratio (1:1) renders best — the existing creatives are already 1:1

## After dropping the files

Refresh the browser. No code changes are needed — the placeholder will be replaced by your image automatically.
